# Rivstephane-RiviereStephane_7_24092021

Ces logiciels doivent être installés :

MysQL : --crées une bdd  et garder le nom, le user et le mots de passe pour les fournir a sequelize
NPM

Démarrer le projet :

dans le dossier Frontend
Ouvrir un terminal dans le dossier et faire : npm install
ensuite : npm run serve

dans le dossier Backend
Ouvrir un terminal dans le dossier et faire : npm install
Ouvrir le fichier config/config.json et remplacer les information de la bdd par les les informations requises (le Nom de la BDD, l'utilisateur et le MDP)
faire: serquelize db:migrate
Faire ensuite dans le terminal : node server
Vous pouvez aller sur http://localhost:8080/
